# Paper Snapshot

This directory contains the current public snapshot of the second ACM paper
manuscript used by this repository.

Included files:

- `main.tex`: current LaTeX manuscript
- `main.pdf`: compiled manuscript snapshot
- `references.bib`: bibliography used by the manuscript
- `EVIDENCE_INDEX.md`: mapping between claims, figures, and evidence assets

This snapshot is intentionally lightweight. The canonical working paper source
continues to be refined in the main research repository, while this directory
keeps the public audit repository aligned with the manuscript state referenced
by the frontend, reproducible core, and evidence package.
