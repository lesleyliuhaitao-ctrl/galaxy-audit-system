## Paper II Audit Pipeline Snapshots

This folder preserves the main procedural scripts used to generate the second
paper's audit results in the research workspace.

These scripts cover the core audit chain:

- full-sample residual pathology audit
- pathology sequence figure generation
- MOND-resistant metadata extraction
- distance-edge surrender analysis
- holdout40 stellar-normalization audit
- hard31 gas-gradient and curvature fingerprints
- hard31 topology and reference-family audit
- `Vgas` spectrum comparison

These files are included as provenance-preserving research snapshots. They were
copied from the active `ACM_Project` workspace so that the public repository
contains not only final evidence tables but also the procedural audit scripts
behind them.

Important note:

- some scripts still reflect the original research-workspace import structure
- path cleanup may still be needed for fully standalone execution inside this
  public repository

The immediate purpose of this folder is transparency of method, not yet final
API polish.
