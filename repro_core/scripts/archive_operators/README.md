## Archived Audit Operators

This folder preserves the operator-test scripts referenced by Appendix A of the
second paper.

The scripts here correspond to no-new-parameter ACM-side rescue attempts that
were tested against the `hard31` subset, including:

- holographic diffusion
- self-shielding
- nonlinear impedance coupling
- operator generalization checks

They are kept in the public repository because the paper's argument depends not
only on successful audit outputs, but also on a reproducible body of negative
results.

As with the main audit pipeline snapshots, these scripts are preserved from the
active research workspace and may still require path cleanup for fully
standalone execution. Their presence here is primarily for transparency of the
archived operator record.
